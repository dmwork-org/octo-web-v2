# 贡献者行为准则

## 我们的承诺

作为成员、贡献者和维护者，我们承诺让社区参与成为无骚扰体验，不因年龄、体型、可见或不可见残障、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济状态、国籍、外貌、种族、宗教、性取向等因素而区别对待。

我们承诺以开放、友好、多元、包容和健康的方式交流与协作。

## 行为标准

有助于积极社区环境的行为包括：

- 对他人展现同理心和善意。
- 尊重不同意见、观点和经历。
- 给予并接受建设性反馈。
- 对自己的错误承担责任，并向受影响的人道歉。
- 关注整个社区的共同利益。

不可接受的行为包括：

- 使用性化语言或图像，以及性关注或性骚扰。
- 恶意挑衅、侮辱性或贬损性评论、人身攻击或政治攻击。
- 公开或私下骚扰。
- 未经明确许可发布他人的私人信息。
- 其他在专业环境中可被合理视为不适当的行为。

## 执行责任

社区维护者负责解释并执行可接受行为标准，并会针对不适当、威胁性、冒犯性或有害行为采取适当且公平的纠正措施。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人以社区代表身份出现在公共空间时的行为。

## 执行方式

如需报告辱骂、骚扰或其他不可接受行为，请联系社区维护者：

```text
conduct@mininglamp.com
```

所有投诉都会被及时且公平地审查和调查。
