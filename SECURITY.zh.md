# 安全策略

## 报告安全漏洞

请不要通过公开 GitHub issue 报告安全漏洞。

请通过邮件报告安全问题：

```text
security@mininglamp.com
```

你应在 72 小时内收到回复。如果没有，请继续通过邮件追问。

请尽量提供以下信息：

- 问题类型，例如 XSS、注入、鉴权绕过或敏感信息泄露。
- 受影响的 branch、commit 或 release。
- 受影响的源码文件或路由。
- 复现步骤。
- PoC，如有。
- 影响范围和建议修复方式，如有。

## 披露流程

我们采用协调披露流程。请给 maintainer 合理时间调查并发布修复后，再公开披露问题。

修复可用后，我们可能发布 GitHub security advisory，并在报告者同意的前提下致谢。
